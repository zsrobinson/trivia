Trivia Game




About This Game

Random Trivia Questions will appear on screen. According to whichever team's scoreboard is bolded with > < surrounding it will answer. After answering the question, the host will click the question to reveal the answer. At this point, the host presses the check if they got it correct, the x if they got it wrong, or the arrow if they don't believe it was a fair question and would like to skip it.
